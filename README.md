Search, Sort and Filter Functionality

## Overview

This project was developed as part of the Edutech Solution MERN Stack Development Internship (Task 14).

The application demonstrates advanced data retrieval techniques using MongoDB and Mongoose, including:

* Product Search
* Category-based Filtering
* Price Sorting
* MongoDB Query Operations

## Technologies Used

* Node.js
* Express.js
* MongoDB
* Mongoose
* Postman


## Learning Outcomes

* Understanding MongoDB Queries
* Implementing Search Functionality
* Applying Sorting Logic
* Filtering Data by Categories
* Building REST APIs with Express.js
* Working with Mongoose Models

## API Endpoints

| Method | Endpoint                                  | Description            |
| ------ | ----------------------------------------- | ---------------------- |
| GET    | /api/products/seed                        | Insert sample products |
| GET    | /api/products/search?keyword=laptop       | Search products        |
| GET    | /api/products/filter?category=Electronics | Filter by category     |
| GET    | /api/products/sort?order=asc              | Sort products          |

## Final Outcome

Successfully implemented Search, Sort, and Filter functionality using MongoDB and Mongoose queries, enabling efficient product data retrieval.
